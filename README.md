# nautilus

A minimal terminal chat interface for [Nemotron 3 Nano](https://huggingface.co/mlx-community/NVIDIA-Nemotron-3-Nano-30B-A3B-4bit) running locally via MLX on Apple Silicon.

## Requirements

- Apple Silicon Mac (M1 or later)
- Python 3.11+
- [uv](https://github.com/astral-sh/uv)

## Setup

```bash
uv sync
```

The model (~17.8GB) is downloaded automatically from HuggingFace on first run.

## Usage

```bash
# Start a new chat
uv run main.py

# Hide the <think> reasoning blocks
uv run main.py --no-thinking

# Limit context to last N turns (prevents memory growth)
uv run main.py --max-turns 20

# Resume a previous chat
uv run main.py --resume chats/2026-03-02_cuda_programming_intro.json
```

Chats are saved automatically to `chats/` on exit as `YYYY-MM-DD_<title>.json`,
where the title is generated by the model. When resuming, you can overwrite the
original file or save as a new one.

Type `quit`, `exit`, or `q` to end a session.

### Chat file format

```json
{
  "meta": {
    "title": "solar_system_overview",
    "date": "2026-03-02",
    "model": "mlx-community/NVIDIA-Nemotron-3-Nano-30B-A3B-4bit",
    "turns": 4
  },
  "messages": [...]
}
```

## Model

| Property | Value |
|---|---|
| Model | Nemotron 3 Nano 30B-A3B (4-bit) |
| Architecture | Hybrid Mamba-2 + Transformer MoE |
| Active params | ~3.6B per token |
| Context | Up to 1M tokens |
| VRAM | ~17.8GB peak |
| Speed | ~97 tok/s on M4 Pro (48GB) |

## Roadmap

### Agent loop
Nemotron was designed for agentic AI and naturally emits structured tool calls
(ReAct-style JSON) without special prompting. The plan:

- [ ] System prompt defining available tools
- [ ] Parse model output for tool call JSON (`{"action": ..., "action_input": ...}`)
- [ ] Implement tools: web search, file read/write, code execution
- [ ] Feed tool results back as context and continue the loop
- [ ] Loop termination: model emits plain text (no tool call) → done

### Chat UX
- [ ] History summarization (compress old turns instead of dropping them)
- [ ] `chats/` index file for browsing past sessions
